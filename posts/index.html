Hugo搭建博客简单教程

分享使用hugo搭建个人博客的简单教程（个人总结） 下载并安装hugo 从官网下载 配置环境变量 Create a new site hugo new site sitaddress(sitaddress这里填写github用户名.gutihub.io-generator) Add a theme 在当前目录下 git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 到这里下载更多主题 Add Some Content hugo new posts/my-first-post.md 在posts目录中找到相应文件进行编辑 完成后draft状态改为false，修改标题时间 Start the Hugo server hugo server -D Navigate to your new site at 你的博客预览. Customize the Theme 在config.toml中 修改title命名标题 修改baseURL定义你的域名 想看更多的定制方法，请看这里 Build static pages hugo
